# Redefinindo Senhas Padrão

Este guia explica como alterar senhas pré-configuradas em sistemas recém-instalados.

1. Acesse o painel de administração digitando o IP do servidor no navegador.
2. Entre com as credenciais padrão informadas pelo fabricante.
3. Procure a seção **Segurança** e clique em **Alterar senha**.
4. Insira uma nova senha forte contendo letras maiúsculas, minúsculas, números e símbolos.
5. Salve as alterações e faça logoff para testar a nova combinação.

Mantenha as senhas em local seguro e jamais reutilize combinações antigas.
