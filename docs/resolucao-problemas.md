# Resolução de Problemas Comuns

Antes de abrir um chamado de suporte, verifique estas etapas:

1. Reinicie o computador ou dispositivo afetado.
2. Certifique-se de que todos os cabos estejam conectados corretamente.
3. Verifique se há atualizações pendentes do sistema ou do aplicativo.
4. Consulte os logs de erro para identificar mensagens específicas.
5. Caso o problema persista, registre o passo a passo que leva ao erro e envie ao suporte.
