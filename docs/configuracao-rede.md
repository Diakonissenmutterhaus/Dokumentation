# Configuração Rápida de Rede

Siga este passo a passo para conectar seu dispositivo a uma rede local:

1. Conecte o cabo Ethernet ou escolha a rede Wi-Fi desejada.
2. Abra as configurações de rede do sistema operacional.
3. Defina se o IP será atribuído automaticamente (DHCP) ou manualmente.
4. Caso manual, informe IP, máscara de sub-rede, gateway e servidores DNS.
5. Salve as alterações e teste a conexão acessando um site qualquer.

Se encontrar problemas, reinicie o roteador e verifique se os cabos estão corretamente encaixados.
