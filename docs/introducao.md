# Introdução

Bem-vindo à sua documentação! Aqui você explica o propósito, para quem é, etc.

Este projeto serve como exemplo de organização de documentação utilizando
arquivos Markdown. A ideia é manter cada assunto separado em um arquivo para
facilitar a manutenção e futuras expansões.

## Objetivos

- Demonstrar uma estrutura simples de navegação
- Permitir que novas páginas sejam adicionadas rapidamente
- Servir como ponto de partida para pequenos projetos
